# Nested

It can be nested.

```html
<n-collapse>
  <n-collapse-item title="right" name="1">
    <n-collapse>
      <n-collapse-item title="right" name="1">
        <div>good</div>
      </n-collapse-item>
      <n-collapse-item title="right" name="2">
        <div>good</div>
      </n-collapse-item>
    </n-collapse>
  </n-collapse-item>
  <n-collapse-item title="right" name="2">
    <div>good</div>
  </n-collapse-item>
</n-collapse>
```

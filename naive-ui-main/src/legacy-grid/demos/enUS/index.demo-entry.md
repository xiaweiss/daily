## API

### Row Props

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| gutter | `number \| string \| [number, number] \| [string, string]` | `0` | `horizontal gutter` or `[horizontal gutter, vertical gutter]` |

### Col Props

| Name   | Type     | Default | Description |
| ------ | -------- | ------- | ----------- |
| span   | `number` | `1`     |             |
| offset | `number` | `0`     |             |
| push   | `number` | `0`     |             |
| pull   | `number` | `0`     |             |

# UpdateValue Event

Why make the `update:value` event an example? Because there's so little to write.

```html
<n-space vertical>
  <n-select
    placeholder="Please Select a Song"
    :options="options"
    @update:value="handleUpdateValue"
  />
  <n-select
    multiple
    placeholder="Please Select Songs"
    :options="options"
    @update:value="handleUpdateValue"
  />
</n-space>
```
